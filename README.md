# Wisconsin Data Center Risk Analysis

## Overview

This project presents a GIS-based spatial analysis of risks associated with data center development in Wisconsin. It integrates climate, water, infrastructure, and social factors into a composite risk framework.

## Data

* Data center locations (operational, proposed, cancelled)
* Flood exposure and climate hazard data
* Water stress indicators
* Grid and transmission infrastructure
* Social conflict indicators
